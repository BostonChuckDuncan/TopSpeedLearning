Hunter Tutorials

Online Tutorials available after login are being written to guide new users through the steps of setting up basic and advanced learning sessions.

Tutorials consist of two parts: Programmed Learning and Recall Exercises

Programmed learning is a document (or lecture), which presents material - in this case - about how to use both types of learning in TopSpeed. Users fill-in-the-blanks for material that is currently or was recently presented. Based on the responses, PL can repeat previous material possibly with greater detail than was shown the first time, or it can skip forward in the lecture.

Recall exercises randomly choose questions about a subject and repeat them just often enough to assure the user is able to recall the correct answer. TopSpeed waits longer and longer times with each correct answer based on the individual user.

The Recall "Review" option used later the same day and perhaps a day or so later, can be assure having firm recall.
